Welcome to ICSINC

The First International Conference on Signal Processing on Information, Network and Computer (ICSINC) focuses on the key technologies and challenges of signal processing on information theory, network application, computer theory and application and so on.

The research areas are convergence by the similarity with both algorithms and applications. The current state and trend of signal processing will be exchanged in this conference. This conference solicits both original research and tutorial articles that investigate the key technologies in various scenarios.
